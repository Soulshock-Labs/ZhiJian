## 第一段：产品功能及核心应用场景（0:00–0:40）

**画面：品牌封面 + 幼师工作场景混剪/插画**

**旁白：**
> 中国 4600 万幼师中，超过 60% 的时间被消耗在文案、模板、记录和返工上——而不是陪伴孩子。
> 纸笺幼师，不做教育 AI 聊天工具，只做一件事：把幼师每天重复、分散、低效的文案任务，变成可执行、可追踪、可导出的 AI 工作流。
> 周计划、日教案、观察记录——输入一个主题，AI 生成完整方案，一键导出 Word，直接用于教学。

**画面切换：**
- 0:00–0:10 幼师伏案工作的痛点画面（或插画）
- 0:10–0:25 产品 Logo + SLOGAN 出现
- 0:25–0:40 三个核心场景图标依次浮现：周计划、日教案、观察记录

**字幕（同步）：**
```
纸笺幼师 · AI 幼师工作流
周计划 | 日教案 | 观察记录 | 一键导出
```

## 第二段：核心功能演示（0:40–1:40）

### 场景 1：工作台首页（0:40–0:55）

**画面：录屏进入 zhijian.me 工作台**

**操作：**
1. 浏览器打开 `https://zhijian.me`
2. 展示工作台首页：周计划卡片、日教案卡片、观察记录卡片
3. 鼠标 hover 到"本周周计划"卡片

**旁白：**
> 打开纸笺工作台，本周任务一目了然。上周停在哪里，系统会主动提醒续写。

**字幕：**
```
智能工作台 · 任务主动续写
```

---

### 场景 2：生成周计划（0:55–1:15）

**画面：点击"开始本周周计划" → 进入生成流程**

**操作：**
1. 点击"开始本周周计划"
2. 输入主题："春天来了"、班级：中班
3. 点击生成，展示 AI 生成中的 loading 状态
4. 生成结果展示：周一至周五的完整活动安排
5. 点击"导出 Word"，展示下载提示

**旁白：**
> 输入主题和班级，AI 自动生成完整的周计划。从活动目标到准备材料，从教学过程到延伸活动，全部覆盖。不满意随时重新生成，满意了一键导出标准 Word 文档，直接打印或提交检查。

**字幕：**
```
输入主题 → AI 生成 → 一键导出 Word
```

---

### 场景 3：日教案与观察记录（1:15–1:30）

**画面：切换到日教案页面 + 观察记录页面**

**操作：**
1. 快速展示日教案页面（从周计划自动延续生成）
2. 切换到"观察记录"页面
3. 展示"拍一张或说一句都行"的入口
4. 展示生成后的观察记录文本

**旁白：**
> 日教案从周计划自动延续，无需重复输入。观察记录支持语音和图片——拍一张照片、说一段话，AI 自动整理成规范文本。

**字幕：**
```
日教案自动延续 · 观察记录语音/图片直出
```

---

### 场景 4：多端覆盖（1:30–1:40）

**画面：三分屏展示 Web / 微信小程序 / 家园共育页**

**操作：**
1. 左屏：Web 工作台（桌面端）
2. 中屏：微信小程序（手机端）
3. 右屏：家园共育落地页（家长端预览）

**旁白：**
> 一端生成，多端同步。老师在 Web 端写计划，在小程序随时查看，家长通过家园共育页同步了解。

**字幕：**
```
Web 工作台 · 微信小程序 · 家园共育
一端生成，多端同步
```

## 录制技术建议

| 项目 | 建议 |
|------|------|
| 录屏软件 | OBS Studio（免费）或 ScreenStudio |
| 分辨率 | 1920×1080，60fps |
| 音频 | 耳麦录制，后期用剪映/PR 降噪 |
| 字幕 | 剪映自动识别 + 手动校对 |
| 背景音乐 | 轻柔钢琴或纯音乐，音量低于旁白 20dB |
| 鼠标高亮 | 开启鼠标点击特效（黄色光圈）|
